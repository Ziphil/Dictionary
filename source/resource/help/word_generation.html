単語の自動生成について

単語の綴りをコンピュータに自動生成させることができます。 複数の単語を生成した後に自動的に訳語を割り当てて辞書に追加する方式と、単語データ編集時に単語の綴りを生成する方式の、2通りの方法で利用できます。 単語データ編集時の自動生成は、OneToMany形式でのみ利用できます。

複数の単語の生成

メニューバーから「編集」→「単語の自動生成」を選択すると、生成時の各種パラメータの設定画面が開きます。 ここでの詳しい設定方法については、「ツール/簡易単語生成」および「ツール/Zatlin」の項目を参照してください。

「訳語セット」の欄では、生成した単語に対して割り当てる訳語のセットを選択してください。 「表示」のボタンをクリックすると、選択中の訳語セットの内容を見ることができます。 「空白～単語」となっているものは全ての訳語が空になっているセットで、生成後に自分で訳語を決めたい場合に便利です。

ここで選択する訳語セットは、自由にカスタマイズすることができます。 その方法については、以下の「訳語セットの編集方法」の節をご覧ください。

「OK」をクリックすると単語が生成され、辞書にそのデータが追加されます。 単語の生成には数秒かかる場合もあります。

訳語セットの編集方法

訳語セットのデータは、ZpDICがあるフォルダ内にあるdata/collectionフォルダに置かれています。 1つの訳語セットに対して1つのJSONファイルが対応しています。 JSONファイルの中身については、以下のような構造になっています。

トップレベルオブジェクトはnameとdataの2つのキーをもちます。 nameキーの値はその訳語セットの名前で、ZpDICから訳語セットを選ぶ際に表示されます。 dataキーの値はオブジェクトの配列です。 この配列の各要素は、以下のような構造になっています。 

{
  "equivalents": ["訳語1", "訳語2", "訳語3"],
  "content": "この単語に関する語法などの補足説明"
}

equivalentsキーの値は訳語の文字列から成る配列です。 訳語の個数はいくつであっても構いません。 contentキーの値は文字列で、単語データ生成時に用例欄や内容欄に書き込まれます。 特に書き込む内容がない場合は、nullを指定してください。
