SlimePlainWord

int getId()

IDを返します。 JSONファイルにおけるentryキーのオブジェクトのidキーの値です。

String getName()

名前を返します。 JSONファイルにおけるentryキーのオブジェクトのformキーの値です。 JSONファイルのキー名とメソッド名が異なるので注意してください。

ArrayList<SlimeEquivalent> getEquivalents()

訳語データの配列を返します。 訳語データはSlimeEquivalentオブジェクトです。 JSONファイルにおけるtranslationsキーに対応しています。 JSONファイルのキー名とメソッド名が異なるので注意してください。

ArrayList<String> getTags()

タグの配列を返します。 JSONファイルにおけるtagsキーの配列です。

ArrayList<SlimeInformation> getInformations()

内容データの配列を返します。 内容データはSlimeInformationオブジェクトです。 JSONファイルにおけるcontentsキーに対応しています。 JSONファイルのキー名とメソッド名が異なるので注意してください。

ArrayList<SlimeVariation> getVariations()

変化形データの配列を返します。 変化形データはSlimeVariationオブジェクトです。 JSONファイルにおけるvariationsキーに対応しています。

ArrayList<SlimeRelation> getRelations()

関連語データの配列を返します。 関連語データはSlimeRelationオブジェクトです。 JSONファイルにおけるrelationsキーに対応しています。
