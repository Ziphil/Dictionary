PersonalPlainWord

String getName()

単語名を返します。

String getPronunciation()

発音記号を返します。

String getTranslation()

訳語を返します。

String getUsage()

用例を返します。

int getLevel()

単語レベルを返します。 返される整数値は0以上15以下です。 PDIC上で「なし」を選択している場合は、この値は0になります。

int getMemory()

暗記マークを返します。 返される整数値は0もしくは1です。

int getModification()

修正マークを返します。 返される整数値は0もしくは1です。
